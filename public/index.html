Vibe Coding Workflow

  1. Click the button below to copy the instruction manual for the AI.

  2. Paste the copied text into the AI's chat and wait for it to acknowledge the rules.

  3. Explain your simulation idea to the AI and type: "Create a schema for this simulation."

    Think of a Schema as a strict blueprint—a detailed list of all the pieces we will need.

    Example: "I want a simulation of a hundred glowing bouncy balls falling into a bowl. Create a schema for this simulation."

  4. Copy the schema created by the AI and paste it below.

  5. Click the button below to generate and copy the WGSL Skeleton (the program framework).

  6. Paste the skeleton into the AI's chat so it can write the physics logic.

  7. Copy the finalized WGSL code generated by the AI and paste it below.



  8. Click the button below to start your simulation.
